01.2 Estados y gestionadores de sus cambios

A pesar de que en los componentes se pueden utilizar variables de javascript común a través de const y let, lo correcto es utilizar estados , ya que estos permiten visualizar y computar cambios en vivo durante la aplicación, mediante la monitorización de cambios ReactJs a través de su virtual DOM.

Notación y uso de estados

El uso de un estado se encuentra asociado al hook useState, cuya notación a utilizar en el curso será la siguiente:

Considerando un estado de ejemplo que se desee inicializar y se llame edad 

const [edad, setEdad] = useState(25)

Las observaciones son las siguientes:

Declaración

Los estados deben comenzar declarados por un const

Lectura y modificación

edad es el elemento utilizado para leer el contenido del estado (25) y setEdad es para modificarlo. Para cambiar el contenido del estado (por ejemplo, cambiar la edad a 26) debería utilizarse la siguiente sintaxis:

setEdad(26)
Inicialización

El hook de useState permite inicializar un estado con valores, en este caso, edad inicializará con un valor de 25 en el momento que se declaró el hook como useState(25)

Ejemplo inicial de lectura y modificación de estados a través de funciones gestionadoras (handlers)

Objetivo del ejercicio: Implementar un componente en ReactJs que permita modificar un contador y a su vez desplegarlo en tiempo real

Ubicado en /src/components/ej01_estados/EstadoInicio.jsx
Accesible desde la url: http://localhost:5173/ejemplos/01/estados-inicial

  1. Asegurarse de que se encuentre importado el hook de useState
import { useState } from 'react'
  1. Declaración de estado
const [conteo, setConteo] = useState(0)
  1. Mostrar el contenido del estado en la sintaxis de etiquetas parecida a HTML (Jsx 1 2)
<h1>Ejemplo de estado de conteo, total contabilizado: {conteo}</h1>
  1. Establecer una función gestionadora (handler) que aumente el contador (dentro del componente).
const handleConteoClick = (evento) => {  
	setConteo(conteo + 1)  
}
  1. Incluir el gestionador (handler) en el evento onClick del botón
<button type="button" onClick={handleConteoClick}>Aumentar conteo</button>
