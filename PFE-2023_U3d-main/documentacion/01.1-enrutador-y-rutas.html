01.1 Enrutador y rutas

Para las aplicaciones [SPA](Single-page application - Wikipedia, la enciclopedia libre) es necesario implementar algún administrador de rutas dinámicas que puedan manejar las solicitudes desde las URLs.

Para la unidad se utilizará la librería del enrutador estándar [react-router-dom](Single-page application - Wikipedia, la enciclopedia libre)

Inicio rápido

Se debe verificar que la librería se encuentre instalada en package.json dentro del proyecto.

"dependencies": {  
	...
	"react-router-dom": "^6.14.1",  
	...
},

De no encontrarse instalado, debe realizarse a través de yarn o npm.

yarn add react-router-dom ó npm install react-router-dom

Configuración

Se configurará react-router-dom desde el punto de inicio de la aplicación: main.jsx

  1. Añadir a las importaciones
import {  
	createBrowserRouter,  
	RouterProvider,  
} from "react-router-dom"
  1. Crear un arreglo de objetos de ruta, a través de la función createBrowserRouter
const router = createBrowserRouter([  
	{  
		path: "/",  
		element: <App />,  
	},  
	{  
		path: "/hola",  
		element: <HolaMundo />  
	},  
	{  
		path: "/ciudad",  
		element: <Calama />  
	}  
])

No olvidar importar las páginas / componentes que utilizará para cada ruta

import App from './App.jsx'
import HolaMundo from ''./components/HolaMundo'
import Calama from ''./components/Calama.jsx'
  1. Realizar la envoltura de la aplicación para que trabaje con las rutas
ReactDOM.createRoot(document.getElementById('root')).render(  
	<React.StrictMode>  
		<RouterProvider router={router} />  
	</React.StrictMode>,  
)
  1. Realizar las pruebas correspondientes, con el servidor web ejecutándose y accediendo desde el navegador a las rutas asignadas de ejemplo:
  • http://localhost:5173/ > página de componente <App />
  • http://localhost:5173/hola > página de componente <HolaMundo />
  • http://localhost:5173/ciudad > página de componente <Calama />
Rutas

Personalice el paso #2 según sea la necesidad del enrutamiento.

Ejercicio

Cree un componente llamado "Pais" y una ruta nueva /pais que muestren un contenido de ejemplo acerca de Chile
