01.0 Componente básico

El elemento componente será el primero en utilizarse y corresponde en primera instancia al equivalente de cada página web que se trabajará.

Mas adelante también este concepto se utilizará de forma extendida a subcomponentes o fragmentos.

Plantilla inicial

Para el curso se presenta la primera plantilla de un componente vacío de la siguiente forma:
Archivo: NombreComponente.jsx

// core - react
import React, { useState, useEffect } from 'react'  
  
// View  
const NombreComponente = () => {  
  
	return (  
		<>  
			<h1>Contenido del componente</h1> 
		</>  
	)  
}  
  
export default NombreComponente
Almacenamiento

Se recomienda siempre trabajar los componentes en /src/components, y los subcomponentes (o también llamado fragmentos) en /src/components/fragments

Notación

Los nombres de los componentes y de su archivo correspondiente se trabajan en notación PascalCase

Ejercicio

Cree un componente nuevo que muestre un título y un párrafo de contenido seleccionado por usted, llamado "Informacion"
